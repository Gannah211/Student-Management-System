# Student Management System

A simple web-based Student Management System built using **PHP** and **SQL Server**.  
The system allows users to manage student records through basic CRUD operations.

## Features

- Add new students to the database
- View a list of all students
- Edit existing student information
- Delete student records
- Display department names using SQL JOIN

## Technologies Used

- PHP
- SQL Server
- HTML
- CSS
- XAMPP

## Project Structure

Student-Management-System
│
├── index.php       # Displays student list
├── add.php         # Add new student
├── edit.php        # Edit existing student
├── delete.php      # Delete student
├── db.php          # Database connection
├── style.css       # Styling index
├── addPAge.css     # Styling add
└── editStyle.css   # Styling edit

## How to Run the Project

1. Install **XAMPP**.
2. Start **Apache**.
3. Place the project folder inside:
   htdocs
4. Create the database and tables in **SQL Server**.
5. Update the database connection inside `db.php`.
6. Open the project in your browser:
   http://localhost/Student Management System
